Living style guide

A modern system for a warm country inn.

Neutral canvas, big confident type, generous whitespace, and rounded photo-first modules — a full modern redesign. Warmth comes from a single terracotta accent and the photography itself. Every token and module below is production CSS from design-system.css.

Colors

A warm-grey neutral ladder does the heavy lifting. Terracotta clay is the only chromatic accent — used for eyebrows, prices, links, and active states. Ink is both primary text and the primary button.
